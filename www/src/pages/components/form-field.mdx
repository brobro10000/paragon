# FormGroup

<SingleComponentStatus frontmatter={props?.pageContext?.frontmatter} />

`FormGroup` and the suite of components outlined here are the foundation for the
to be developed, `TextField`, `SelectField`, and `TextareaField`.

FormGroup renders its children inside a context that provides ids and attributes
that enable these components to relate to each other according to WCAG guidelines.

##### Basic usage with floating label

```jsx live
() => {
  const [value, setValue] = useState('');
  return (
    <FormGroup>
      <FormControl
        leadingElement={<Icon src={FavoriteBorder} />}
        trailingElement={<Icon src={Verified} />}
        floatingLabel="What kind of cats?"
        value={value}
        onChange={(e) => setValue(e.target.value)}
      />
      <FormControl.Feedback>We answer the important questions here.</FormControl.Feedback>
    </FormGroup>
  );
}
```

##### States

```jsx live
() => {
  const [value, setValue] = useState('');
  const handleChange = (e) => setValue(e.target.value);
  return (
    <>
      <FormGroup isInvalid>
        <FormLabel>What kind of cats?</FormLabel>
        <FormControl value={value} onChange={handleChange} />
        <FormControl.Feedback type="invalid">
          You are incorrect!
        </FormControl.Feedback>
      </FormGroup>

      <FormGroup isValid>
        <FormLabel>What kind of cats?</FormLabel>
        <FormControl value={value} onChange={handleChange} />
        <FormControl.Feedback type="valid">
          You are correct!
        </FormControl.Feedback>
      </FormGroup>

      <FormGroup>
        <FormLabel>What kind of cats?</FormLabel>
        <FormControl value={value} onChange={handleChange} />
        <FormControl.Feedback>
          Let me help you!
        </FormControl.Feedback>
      </FormGroup>

      <FormGroup>
        <FormLabel>What kind of cats?</FormLabel>
        <FormControl value={value} onChange={handleChange} />
        <FormControl.Feedback icon={<Icon src={WarningFilled} />}>
          Making it custom. Be very careful.
        </FormControl.Feedback>
      </FormGroup>

      <FormGroup>
        <FormLabel>What kind of cats?</FormLabel>
        <FormControl value={value} onChange={handleChange} />
        <FormControl.Feedback>
          More than one at a time!
        </FormControl.Feedback>
        <FormControl.Feedback type="valid">
          This is okay sometimes.
        </FormControl.Feedback>
      </FormGroup>
    </>
  );
}
```
